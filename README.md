# TRbeater
The goal of this simulation is observation of the formation of social circles with simple rules. There is a Turkish "çizelge.txt" file in the project. It's actually
terrible for a global project but my friend insisted I post it on github. Anyways, I think of adding UI elements and more fixes to physics, it is 64 bits only and 
requires OpenGL 4.3 or higher. It uses a compute shader to brute force calculation of forces between different people. But at the time of 09 July 2026 it's general
enough to be integrated into a particle physics engine. I use Code::Blocks for the project and would really be happy if anyone could make it into a CMake project.
